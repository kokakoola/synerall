Syneralli teema

Aluseks Bootstrap 3 ametlik sass-versioon, mis on töö nõuetele vastavalt ainult desktop-versioonis.

Projekti kompileerib Hammer for Mac.

Teema Rapido on rikkalik - liiga. Praegu proovin teda nudida meie vajadustele vastavaks, sh töötama sassiga ja ilma compassita.
